# About
This contains a mirror of the main project, except it was downgraded to LTS22 via ProjectTool. This is to ensure that packages are made for the lowest possible GM version and won't have any issues importing! Woo :D

# Usage
If needed, run the "downgrade.bat" file to perform a downgrade using PackageTool. The path is hard-coded so ensure it is properly setup for your environment! (GameMaker-Beta is needed)
